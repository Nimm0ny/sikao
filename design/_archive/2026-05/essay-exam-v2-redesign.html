申论考场 V2 — UI 优化原型

按你 v0 反馈调整：1 圆环修几何（保留 metaphor），2 材料 tabs 保留 before + 加横向滚动，3 抽屉用 after，4 字号给 PM 建议（建议删），5 报告页继续优化（去掉所有斜体强调，加优先级 / 维度对比表 / 行动建议 / 你的 vs 范文 双栏对比）。

1 题目按钮

保留圆环 metaphor，修几何关系：内圆 r=10 stroke=2.5（占 8.75–11.25），外圆 r=14 stroke=1.5（占 13.25–14.75），2px 间隙分得清字数主进度 vs 时间副进度。Active 时 capsule 扩展显示题号·题型·进度 inline 文字（保留 before 行为）。

2 材料 tabs

保留 before 视觉（flex 平铺 + ink→accent 渐变下划线 + 黄底 hits + 蓝 dot），加横向滚动：每 tab 固定 84px，超过容器自动 overflow，两端 fade mask + chevron 按钮提示。当前选中 tab 自动 scroll 到视野中间。7 篇够用。

3 抽屉按钮

用 after：collapse 按钮并入 panel header（搜索 / 总览 / 收起 三联），不再浮右上角。折叠态保留 80px vertical strip "点击查看给定资料"。

4 字号

PM 建议：删掉。95% 用户不会主动调；浏览器原生 ⌘+ / ⌘− 兜底；申论考场专注时间紧，UI noise 越少越好。toolbar 只留"划线"toggle。
替代方案：挪去 panel header 的 "总览" 旁边再加个齿轮按钮，点开抽屉里放字号 + 行距 + 阅读模式。但更值得做的是别的功能（见下方 PM backlog）。

5 报告页

全部去 italic（项目规范）。4 段结构：Hero band（96px 总分 + 5 维雷达 + 最该改 callout）→ 题目分项 strip（加"建议先改"优先级标）→ 新 全卷 5 维对比表（按问题严重排序 + 历史 sparkline + Δ 值）→ 新 学习行动 3 个 action items + 关联同类题练习。Scene D 单题展开改 双栏：你的 vs 范文 同段对照，弱句/亮句段落级标注。

RECOMMENDATION

删掉 toolbar 上的字号控件

申论考场专注时间紧 (≥ 60min)，UI noise 越少越好。toolbar 只留"划线"toggle，主路径更聚焦。

删掉的理由 PRIMARY

① 95% 用户不会主动调字号（行业经验）；
② 浏览器原生 ⌘ + / ⌘ − 已经能解决（虽然影响整页缩放，但桌面端没问题）；
③ 当前 toolbar 上"划线"+"划线 N 处" 已占左半，加 3 段字号控件视觉很挤；
④ 真有视力问题的用户会主动找设置，不会指望 toolbar。

替代方案（备用）

panel header 已有 3 个 iconbtn（搜索 / 总览 / 收起）。如果坚持要给字号入口，再加 1 个齿轮按钮，点开抽屉里放 字号 + 行距 + 阅读模式 三档预设（紧凑 / 标准 / 宽松），比单字号有用。

更值得做的（PM backlog）

同等优先级的 5 项功能，每个都比"调字号"对申论用户更有价值：

  • 划线分类 论点 / 论据 / 问题 / 对策 4 类，划线时选 → 一键导入答题卡或草稿
  • 段落自动续编 检测"第一，"自动建议"第二，""第三，"，申论刚需
  • 字数实时统计 每段、每句字数浮显（应对题逐段控字数）
  • 疲劳水印 60min 后弹微提示"喝口水"，纯休息不停表，UX 体感
  • 答题卡缩略 右下角 mini-map 显示 1000 字网格分布，长文跳定位
  • 关键词 chip 从划线段自动生成关键词 chip，作答时浮显 chip 提示

能源结构的每一次重塑，本质上都是治理理念的升级。从 "够不够用" 到 "用得绿"，再到 "用得智"，电源体系的演进折射出我国从粗放管理迈向精细治理的轨迹。

当 2025 年单月用电量首破 1 万亿千瓦时，"装机已够、负荷敢迎" 的从容背后，是一张由风、光、水、核多源协同的新型电力网。资料 4 揭示出三条主线：清洁能源占比提升、骨干电网联通增强、智能调度精度跃升。

清洁能源占比的跨越，需要更高的统筹定力。仅靠单一项目堆叠难成体系，须坚持顶层设计先行——从国家电网规划层面打通跨区域绿电流转通道。

电网的弹性来自数据的颗粒度。每秒级的负荷预测、毫秒级的故障切换，都依赖跨场景的数据互通。
