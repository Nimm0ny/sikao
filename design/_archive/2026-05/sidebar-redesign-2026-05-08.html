决策记录 (D1-D4)

D1 — 路由策略 (零变更)

  • 不动 frontend/src/router/index.tsx, sidebar 只改导航语义.
  • 首页 → /app (整合 hub 保留).
  • 行测·专项练习 → /categories (现 6 大类入口).
  • 行测·套卷练习 → /app#paper-list (锚点滚到 Home 底部 PaperListSection).
  • 申论·专项练习 → disabled "敬请期待" (无真实 view, fail-fast 不指虚假 query).
  • 申论·套卷练习 → /essay/papers (745 套申论真题).

D2 — 折叠/展开

  • 默认全部 group 展开, 让新用户立即看见全部入口.
  • 点击 group label (button) → toggle, chevron 旋转 90° (展开 0deg / 折叠 -90deg).
  • localStorage key sidebar-groups-collapsed = ['xingce','essay'] 切片. 默认空数组.
  • active 子项即使在折叠 group 内, 也会自动展开父级 (避免用户点回来时找不到).

D3 — 视觉

  • 沿用 sidebar v2 的 deep ink (--sidebar / --sidebar-fg-* token), 不引新颜色.
  • Group icon 用 inline SVG (行测=三横线表格 / 申论=文档+横线), 子项无 icon, 用小圆点 + 12px 缩进保持视觉对齐.
  • Active 状态沿用 v2 的 layoutId framer-motion 平滑滑动, 不引新动效.
  • chevron 14×14 SVG, 旋转走 transform + var(--motion-fast).

D4 — a11y

  • Group label 是 <button aria-expanded aria-controls="grp-xingce">, 子列表 id="grp-xingce".
  • Disabled 子项是 <span aria-disabled="true" title="敬请期待">, 不是 <a> (避免 keyboard tab 误入).
  • chevron aria-hidden, group icon aria-hidden; 文字 label 是 a11y 真信源.
