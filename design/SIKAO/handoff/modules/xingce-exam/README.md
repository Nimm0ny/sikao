## Overview

行测答题是一个针对**公考行政职业能力测验**的在线答题界面，覆盖四种官方题型：

1. **单选题** — 一题干 / 4 选项
2. **多选题** — 一题干 / 4 选项 / 多答
3. **判断题** — 一题干 / 2 选项（正确 / 错误）
4. **资料分析** — 一题干材料 + 5 道子题 / 每题 4 选项

设计目标是在长时间高强度做题场景下，保持**纸质阅卷的安静感** + **数字工具的便利**：辅助操作（划线、笔记、收藏、标记、AI 解析）都不打扰主答题流。

## About the Design Files

本包中的 HTML 文件是**设计参考稿**，不是可直接上线的代码。它们是用静态 HTML / CSS / 少量 vanilla JS + 一个 React Tweaks 面板搭建的交互原型，用来表达**预期的视觉与交互**。

落地任务是：**在目标代码库（React / Vue / Next.js / 小程序 / 移动端等）中按现有架构 + UI 库重建这套设计**。若项目尚无前端环境，请按团队习惯选型实现；保持视觉与交互的等价即可，不必逐行翻译 HTML。
