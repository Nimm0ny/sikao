## 关键交互
- 大分数动画从 0 → 实际分（800ms ease-out）
- weak 行末尾的暗朱小圆点点击 → 展开"为什么弱 + 推荐题集"
- "下一步"侧栏三按钮：`再来一次` / `导出 PDF` / `加入计划` (+ 写入 plan 后弹 toast)
- 申论 Q3 weak 行 (72%) 默认展开，引用准确度 62% 红字

## 数据
```ts
type ResultVM = {
  attempt: Attempt;
  breakdown: { name: string; total: number; correct: number; timeSec: number; isWeak: boolean }[];
  ai: { recap: string[]; plan: PlanItemDraft[] };
  // 申论
  essayGrades?: { qid: string; total: number; rows: { name: string; got: number; max: number; weak?: boolean }[] }[];
  citationAccuracy?: number;
};
```

## 验收
- [ ] 大数字 serif tabular，小数点后一位
- [ ] 表格不要 zebra；只有 weak 行有暗朱左侧 1px border
- [ ] "导出 PDF" 真能 `window.print()` 出干净版本
- [ ] 申论 Q3 默认展开，其余折叠
