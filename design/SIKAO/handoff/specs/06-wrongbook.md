# 06 · 错题本

> HTML 锚点：`06 · 错题本` · 路由 `/wrongbook`

## 布局
```
┌───────────────────────────────┬──────────────┐
│ 顶部筛选条                    │ 侧栏：        │
│  全部 (148) · 行测 · 申论 ·   │  统计        │
│  按错的次数 ↓ · 已掌握 (32)   │  到期复习 12 │
├───────────────────────────────┤  按模块分布   │
│ 错题列表（卡片堆叠）           │              │
│  - 题干 / 我的答案 / 正确答案 │  操作         │
│  - 错次 ×3 · 上次 5月7日       │  · 开始复盘  │
│  - tag: 资料分析 · 比重计算    │  · 全部标记  │
│  - 折叠的解析 / AI 复盘        │              │
└───────────────────────────────┴──────────────┘
```

## 关键交互
- 单卡 hover 整张变 paper-2，右上出 `复盘 →` 按钮
- 标题右侧的 `× 3` 是错次徽章；`已掌握` 一旦标记就移到尾部并变灰
- 顶部筛选用 Chip 多选；筛选条 sticky
- 卡片可拖到侧栏"复习清单"凑成一组，凑够 5 道后启动复盘

type WrongBookVM = {

## 验收
- [ ] 大量错题时（>50 张）虚拟滚动
- [ ] 解析展开是 height 0 → auto 平滑过渡，不要 `display: none` 跳变
- [ ] "已掌握"卡片透明度 0.5，但点击仍可恢复
