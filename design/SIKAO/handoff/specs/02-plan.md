# 02 · 学习计划

> HTML 锚点：`07 · 学习计划` · 路由 `/plan`

## 目标
长周期学习路径管理：周日历 + 今日清单 + 薄弱补强 + 模考节奏。不做花哨甘特图。

## 布局
```
┌──────────────────────────────────────────────────┐
│ EYEBROW · 我的备考计划                            │
│ H1 距 2026 国考 还有 78 天                        │
├──────────────────────────────────────────────────┤
│ 周日历（7 天 · 当前周）                           │
│  Mon  Tue  Wed  Thu  Fri  Sat  Sun               │
│  ●●●  ●●○  ●○○  ─    ─    ─    ─                │
├──────────────────────────────────────────────────┤
│ 左 2/3：今日 / 本周任务清单                       │
│   - 上午 行测 · 资料分析 · 30 题 · 60min          │
│   - 下午 错题复盘 · 15 道                         │
│   - 晚 申论真题 · 副省级 · Q3                     │
│                                                  │
│ 右 1/3：薄弱模块 (3 行) + AI 调整建议             │
└──────────────────────────────────────────────────┘
```

## 关键交互
- 周日历点击切换"查看哪一天"
- 任务行支持拖拽改时间段（hover 显示 `⋮⋮`）；status: 未开始 / 进行中 / 已完成
- "AI 建议"只显示一句，点击展开三条建议（带 ✓ 接受 / ✕ 忽略）
- 顶部"距 D-78"右侧有 `编辑目标` 按钮 → 弹窗修改考试日期
