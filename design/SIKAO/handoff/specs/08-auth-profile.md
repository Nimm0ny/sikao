# 08 · 登录注册 / 个人中心

## 登录 / 注册（HTML 锚点：`01 · 登录 / 注册`）
路由：`/login` `/register`

### 布局（1280×720，居中卡片）
```
┌──────────────────────────────────────┐
│  左：品牌区                           │
│   Mark + SIKAO + 一句 slogan          │
│   底部小字：3 行客户评价 / 数据点      │
│                                      │
│  右：表单                             │
│   H2 登录 / 创建账号                  │
│   - 微信扫码 (大按钮)                 │
│   - 手机号 + 验证码                   │
│   - 邮箱密码                          │
│   底部：协议 + 切换 注册/登录          │
└──────────────────────────────────────┘
```

### 验收
- [ ] 微信扫码占位用线性二维码占位图 (svg)，不要 emoji
- [ ] 协议链接是 `<a class="link">`，下划线 underline-offset 3px
- [ ] 表单 input 高 40，无圆角，focus 边框变 ink

## 个人中心（HTML 锚点：`08 · 个人中心`）
路由：`/profile`

### 布局
```
┌────────────────────────┬──────────────┐
│ 头像 + 姓名 + 计划 chip │ 订阅 (Pro)   │
│ 报名信息（考公时间）    │  到期日       │
├────────────────────────┤  续费按钮     │
│ tabs: 设置 · 偏好 ·     ├──────────────┤
│       数据导出 · 通知   │  快捷入口     │
│ 内容区（按 tab 切换）   │  · 错题导出   │
│                         │  · 学习日志   │
└────────────────────────┴──────────────┘
```

### 关键交互
- "Pro" chip 暗朱底；"Free" chip 描边 ink
- 设置项分组：账号 / 隐私 / 主题 / 通知
- 主题 group 里直接给 3 个 swatch（reading/pure/night），点击即应用并写入 `data-theme`
- 数据导出：CSV / PDF 两按钮

### 验收
- [ ] 切 tab 不刷整页，URL 跟 `?tab=preferences`
- [ ] 退出登录是 `Button.ghost`，描边暗朱
