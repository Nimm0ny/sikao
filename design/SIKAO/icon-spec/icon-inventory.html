Composite · NumberCircle
题号圆

5 states · 3 sizes unanswered · answered · marked · current · wrong
FbDock / FbOpt / FbGroupTabs

复合编号原型 — 数字 + 圆 + 状态 三位一体。本系统传播范围最广（每屏 35-40 个），任何视觉漂移都会立刻被发现。 底层由 frontend/src/icons/composite/NumberCircle.tsx 提供，5 状态切换通过 status prop。

unanswered
第 N 题, 未答, 点击跳转
answered
第 N 题, 已答, 点击跳转
marked
第 N 题, 已标记, 点击跳转
current (中间带 halo)
第 N 题, 当前题
wrong (review)
第 N 题, 答错, 点击查看

FbDock 实战示例 — 35 题答题卡, 5 状态混合
