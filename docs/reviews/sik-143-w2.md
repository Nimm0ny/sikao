### F-01 · P3 · RecentPractice `.list` 用 `overflow:hidden` 而非 `overflow-y:auto`

- 证据：`RecentPracticeSection.module.css:38-48` `.list { ... overflow: hidden }`。

- 建议：保持现状；若未来该卡条数可变，再切 `overflow-y:auto`。

### F-02 · P3 · `.refreshBtn` 圆角用 `--radius-10` 而非契约暗示的方形 icon-btn

- 证据：`RecommendationSection.module.css:38` `border-radius: var(--radius-10)`。
- 说明：契约 §2.1 称「36×36 方形 + 圆角」，`--radius-10`(10px) 即圆角方形，与原型 topbar `.icon-btn`
  圆角同款，符合。仅登记，无需处理。

### F-03 · P3 · `as Error | null` 类型断言（既有，非本次新增）

- 证据：`RecommendationSection.tsx:80` `(query.error as Error | null)?.message`。
- 说明：为既有写法（W2 前已存在），用于 react-query `error: unknown` 的消息提取。非 `as any`，
  属可接受的窄化。建议后续统一走 error 归一化工具，但不在本契约 Scope。

### F-04 · P3 · `Home.test.tsx` 整文件 CRLF 噪声放大 diff
