## Round 1 Findings

1. `medium`
   - 问题：双行 chip 上线后，month / week / DnD 的 `cardLimitPerCell` 仍按单行高度预算，导致可见行数被提前裁切。
   - 处理：统一把窗口预算从 `space-6` 收到 `space-8`，并同步 month / week / DnD 的断言。

2. `medium`
   - 问题：默认 mock / 测试把 `linkedSessionId: null` 和 `not_submitted` 绑在一起，导致“本该是未关联”被假通过成“未提交”。
   - 处理：默认 Home mock 改成 `missing_linked_session -> 未关联`；测试里只有真的有 `linkedSessionId` 时才断言 `未提交`。

3. `low`
   - 问题：query error / missing item 时，chip / peek 会静默消失。
   - 处理：共享 hook/state 新增显式 `聚合不可用 / 聚合缺失` 分支，并补测试覆盖。

4. `low`
   - 问题：DnD month grid 虽然透传了 aggregate，但缺少回归测试。
   - 处理：新增 DnD 路径 aggregate 文案断言。

## Round 2 Findings

- 无 findings。
