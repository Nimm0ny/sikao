## 0. 范围 & 约束（最小集）

仅以下 2 条铁线，其余视觉细则由 design tokens（[[Design-System]]）兜底：

1. **每 tab 默认 view 纵向 1 屏**（不滚动）。**例外**：以下 6 条全屏脱壳路由不受 1 屏约束：
   ```
   /practice/sessions/:id              答题
   /practice/sessions/:id/result       结果
   /practice/sessions/:id/grading      申论批改详情
   /practice/ai-questions/generating   AI 出题等待
   /notes/:id                          笔记编辑
   /q/:id/redo                         题目中枢重做
   ```
   `/q/:id`（Question Hub 非 redo）作为详情页**也豁免** 1 屏约束（决策 V8）。子页（`/review/all` / `/review/insights` / `/profile/goals` 等）视为详情页**豁免** 1 屏约束（决策 V9）。

2. **所有 icon 必为 SVG**，禁止 emoji 与文字符号字形（→ ✕ ⭐ 等统一改 SVG）。导航 icon 必带 `aria-label`，被 `lint:icon-button` 兜底（[[CLAUDE]] §4）。

> 上一轮讨论中"视觉原则 & 共用元素（约束）"被显式删除，本节不重复 design tokens / 大数字 italic / 主色规则等内容——这些由 [[Design-System]] tokens 直接管。

## 1. 视口预算

| 维度 | 值 | 说明 |
|---|---|---|
| Baseline viewport | 1280 × 800 | 桌面通常起步分辨率 |
| RailMini 占用 | 64px 左侧 | icon-only，固定 |
| Content max-width | 1216px | 1280 - 64 |
| Page top padding | 32px | 不含 RailMini |
| **1-screen content height budget** | **~720px** | 800 - 32 上 padding - 32 浏览器 chrome buffer ≈ 720 |

> 决策 V1：以最严视口 1280×800 为 baseline。如要换 1440×900 或更高，全部 wireframe 重新核对。

- 全屏脱壳态（§0 6 条路由 + Question Hub）**隐藏** RailMini，全屏沉浸

## 3. 五个 Tab 默认 View 原型（Web）

各 tab 按 1 屏预算（§1）紧凑布局。下文每屏给：默认布局、loading skeleton、空态、错误态共 4 张图。

下文 ASCII 中：

- `▭ w×h` = 矩形 skeleton 占位
- `○ d` = 圆形 skeleton 占位
- `[icon-xxx]` = SVG icon 渲染槽位（具体 SVG 见 §7）
- `[xxx →]` = 按钮，`→` 内含 SVG `icon-ui-chevron-right`

## 6. 三态共用规则

### 6.1 状态机

```
                        ┌────────────────┐
            进入 view ──→  isPending     │
                        └───────┬────────┘
                                │ 200ms 延迟
                                │ (避免快接口闪烁)
                                ↓
                        ┌────────────────┐
                        │ <SkeletonView/> │
                        │ <SkeletonSection>│
                        └───────┬────────┘
                                │
              ┌─ success(items) ┴── error ─────────┐
              ↓                                    ↓
      ┌──────────────┐                    ┌────────────────┐
      │ 渲染数据      │                    │ <SectionError>  │
      │              │                    │ <ViewError>     │
      └──────┬───────┘                    └────────┬────────┘
             │                                     │
       data.length===0?                          [重试]
             │                                     │
             ↓                                     ↓
      ┌──────────────┐                    ┌────────────────┐
      │ <EmptySection>│                    │ refetch        │
      └──────────────┘                    └───────────────┘

stale-while-revalidate（已有数据 + 后台更新）:
  - 不显 skeleton
  - 不显 error
  - 仅顶栏小 spinner（icon-ui-rotate-ccw rotate animation, 12px ink-3）
```

### 6.2 三态共用组件清单

| 组件 | 用途 |
|---|---|
| `<SkeletonBlock w h r?>` | 矩形占位（带 shimmer 动画，1.4s 循环） |

| `<SkeletonCircle d>` | 圆形占位（头像 / 圆环） |
| `<SkeletonView>` | 整页骨架容器（包含 §4.X.2 全套布局） |
| `<SkeletonSection>` | 单段骨架容器 |
| `<EmptySection>`（已有 D14） | icon + title + hint + 可选 CTA |

### 7.1 命名规范

```

   group   = nav | ui | state | sym | obj
   name    = 语义动词 / 名词，kebab-case
```
