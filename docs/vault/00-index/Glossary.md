# Glossary

| 术语 | 含义 |
|------|------|
| **行测** | 行政职业能力测验，公考客观题部分（单选/多选/判断/填空/图形推理等） |
| **申论** | 公考主观题部分（材料阅读 + 多题文本作答） |
| **Paper / 套卷** | 一份完整考卷，含若干 Section、Block、Question |
| **Section** | 套卷分区（如"言语理解"、"判断推理"） |
| **Block** | 题块（如同材料下的多个小题） |
| **MaterialGroup** | 申论材料组 |
| **Answer Session** | 一次答题会话（created/in_progress/paused/submitted/reviewing/expired/cancelled） |
| **Wrong Book / 错题本** | 错题档案，自动加入做错的题 |
| **Mastery** | 错题掌握度（4 字段：consecutive_correct / total_correct / last_review / next_review） |
| **Smart Review / 智能复盘** | 错题本的复习模式（5 mode：daily / random / weakness / urgent / topic） |
| **Note / 笔记** | 题级 markdown 笔记 + 笔记本社交化 |
| **BYOM** | Bring Your Own Model — 用户自带 LLM API key |
| **Specialty / 专项** | 跨卷单题练习（按 canonical_subtype 筛） |
| **Predicted Score** | 预测分（基于历史答题正确率） |
| **fenbi** | 第三方公考题库平台，sikao 的原始数据来源 |
| **AIPTA** | 内部 AI 批改服务（暂无独立部署） |
