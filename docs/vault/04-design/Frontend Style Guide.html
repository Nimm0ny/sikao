Italic 不是装饰

italic 永远表达「辅助 / 旁白 / 注释」。不准用 italic 当装饰强调 — 强调用 weight 600。

允许用法

  • section 副标题：这是决策不是设计
  • 表格 header 行
  • 统计单位：题 / 分钟
  • 引用：— SIKAO · 2026

禁止用法

  • 正文里强调单词 → 用 weight 600
  • 按钮文字 italic
  • 错误提示 italic（太柔，丢急迫感）
  • 大数据 stat 数值 italic

规则 · 写组件前看

  • 所有 SVG 走 currentColor 描边 · 默认 stroke-width: 1.6 · fill: none。允许例外只在「选中/激活 dot」用 fill: currentColor。
  • viewBox 统一 0 0 24 24。单文件 SVG component 命名 IconXxx.tsx 落在 frontend/src/components/icons/。
  • SVG-only IconBtn 必须带 aria-label。lint:icon-button 强制。
  • 禁止把 emoji 当 icon。lint:no-emoji-as-icon 强制 — 答题/申论/笔记路径硬禁，marketing 长文允许。
  • 互斥状态组（如划线/荧光/批注）走单选 toggle — 内部 state 用 'line' | 'highlight' | 'note' | null 而非 3 个独立 bool。

反面教材 · AI slop 避雷

下面这些每出现 1 处就在偏离品牌 — code review 直接打回
10 条铁线
DON'T 01
渐变背景做"科技感"

SIKAO 是白 + 蓝 + 黑灰。不存在「紫→蓝渐变 hero」。主行动用 accent-1 实底，正文仍走 ink。

DON'T 02
Emoji 当 icon

📚 / 🎯 / 👋 / ✨ — 全部用 SVG 替换或删除。marketing 长文里偶尔出现 OK。

DON'T 03
双感叹号 / 鸡血文案

「冲呀！」「你真棒！」「Go！」— 改成陈述句句号。

DON'T 04
圆角左边色条警示卡

那个「四角 12px 圆角 + 左 4px 蓝边」是 ChatGPT generic — 用我们的 EmptyState 或 toast。

DON'T 05
SVG 凭空画插画

不画卡通人物 / 火箭 / 灯泡。需要图就用 image-slot 留位让用户拖图。

DON'T 06
新色直接写 hex

color: #5a78e0 — 不行。要么 token 要么写 oklch 跟现有色谐音 — 但要先在本规范登记。

DON'T 07
border-radius: 8px / 12px 任意值

7 档之外的值禁。lint:radius-token 会卡 CI。

DON'T 08
view 里散文案

所有 empty / error / toast 走 ui-copy.ts。view 内联文案 = 阻塞 PR。

DON'T 09
数字 stat 没单位 / 没 italic

「132 题」永远是 132 + italic ink-3 「题」。单位裸奔显廉价。

DON'T 10
把 italic 当强调

italic = 旁白。强调用 weight 600。混用就丢语义。
