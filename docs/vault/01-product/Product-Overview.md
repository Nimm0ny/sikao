# Product Overview

> 让备考从刷题变成思考。

## 调性

- **品牌**：SIKAO（思考）
- **核心比喻**：图书馆隔壁桌的同学——安静、靠谱、不打鸡血
- **视觉**：ink-first（暖墨黑），蓝是 accent 点缀

## 核心用户旅程

1. **登录 / 注册**（邮箱 / 手机，可选绑定）
2. **进入 Dashboard**（学习中心：继续上次、今日计划、弱项提醒、考试倒计时）
3. **选择练习**
   - 行测：分类树 / 套卷库 / 自定义练习 / 错题重做
   - 申论：套卷 / 专项 / 整卷模考 / 历史草稿
4. **答题**（Answer Session）
5. **提交并查看结果**（成绩单 + 解析 + 错题入库）
6. **复盘**（智能复盘 / 错题详情 / 笔记 / 计划）

## 端

- **Web**（当前唯一已实现端）
- **Mobile**（响应式 fragment 已有 DashboardMobile，未独立成端）
- **Tablet**（申论考场已识别 device-aware，未独立成端）
- **Admin**（后端 API 已有，前端尚未独立）
